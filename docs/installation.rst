============
Installation
============


Stable release
--------------

To install get_video_length, run this command in your terminal:

.. code-block:: console

    $ pip install get_video_length

This is the preferred method to install get_video_length, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for get_video_length can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Tr4shL0rd/get_video_length

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/Tr4shL0rd/get_video_length/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Tr4shL0rd/get_video_length
.. _tarball: https://github.com/Tr4shL0rd/get_video_length/tarball/master
